LeetCode-435 无重叠区间 #

Solution 1 #

按照右端点考虑区间, 如果当前的区间的左端点大于等于上一个区间的右端点, 则可以选择该区间, 否则删除该区间. 贪心地考虑, 删除该区间比删除上个区间更优.

代码如下:

class Solution:
    def eraseOverlapIntervals(self, intervals: List[List[int]]) -> int:
        intervals = sorted(intervals, key=lambda x: x[1])
        ans = len(intervals)
        cur_r = -inf
        for l, r in intervals:
            if l >= cur_r:
                ans -= 1
                cur_r = r
        return ans
